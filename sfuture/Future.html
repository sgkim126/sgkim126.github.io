Class: Future

C new Future<T>( ) → {void}

Creates a new future with ES6 promise. The new future will be completed when the promise is completed.

M fallbackTo( future: Future.<T> ) → { Future. }

Returns a new future. The new future will be failed when both this future and the passed future is failed. The reason will be the same with this future. The new future holds the same value with this future, if this future is completed successfully. And holds the same value with the passed future, if this future is failed and the passed future is successful.
